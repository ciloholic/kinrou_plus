# 勤労の獅子 Plus

勤労の獅子へ自動ログインするChrome拡張機能です。

## 機能

- ログイン画面で認証情報を自動入力してログイン
- タイムアウトエラー画面や500エラー画面を検知してログイン画面へ自動リダイレクト
- パスワードはAES-GCM方式で暗号化して保存
- ブラウザを閉じると暗号化キーが消去されるため、再起動後はパスワードの再入力が必要(セキュリティ対策)

## インストール

1. 依存関係をインストール
   ```bash
   yarn install
   ```

2. CSSをビルド
   ```bash
   yarn build:css
   ```

3. Chromeで `chrome://extensions` を開く

4. 右上の「デベロッパーモード」を有効にする

5. 「パッケージ化されていない拡張機能を読み込む」をクリック

6. `kinrou_plus` フォルダを選択

## 使い方

1. Chrome拡張機能のアイコンをクリック
2. 法人コード、社員コード、パスワードを入力して「保存」
3. 勤労の獅子のログイン画面にアクセスすると自動でログインされます

## セキュリティ

- パスワードはAES-GCM方式で暗号化されてローカルに保存されます
- 暗号化キーは`chrome.storage.session`に保存され、ブラウザを閉じると自動的に消去されます
- ブラウザ再起動後は暗号化キーが失われるため、パスワードの復号ができなくなり再入力が必要です
